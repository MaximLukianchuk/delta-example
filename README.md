# delta Δ

delta - open source библиотека для быстрого написания простых SPA приложений

![Пример кода]()

### Быстрый старт
```
git clone https://github.com/MaximLukianchuk/delta-example
cd delta-example
npm i
npm run dev
```

### Сборка
```
npm run build
```
